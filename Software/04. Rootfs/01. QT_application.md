# QT application - Play Media - using GLIBC toolchain

## 1. Introduction

- Markdown guide that shows how to build a simple Qt media player app (play audio/video) and cross-compile it with a GLIBC toolchain. This guide covers prerequisites, project source, cross-compile toolchain tips, deployment to target, and common troubleshooting.

### Comparison: uClibc vs GLIBC**

| Criteria | **GLIBC (GNU C Library)** | **uClibc (Micro C Library)** |
|-----------|---------------------------|-------------------------------|
| **Purpose** | Designed for full Linux systems (desktop, server, embedded with rich features) | Designed for lightweight embedded systems with limited resources |
| **Feature set** | Very complete — full POSIX, NPTL threads, locales, NSS, dynamic linking, wide char, realtime support | Reduced — many POSIX or advanced features are trimmed or simplified |
| **Compatibility** | High — fully compatible with Linux and most third-party libraries | Partial — some system or POSIX APIs missing or behave differently |
| **Multimedia / Qt support** | ✅ Excellent — compatible with GStreamer, PulseAudio, X11/Wayland, Qt Multimedia | ⚠️ Limited — many multimedia libs fail to build or run properly |
| **When to use** | When you need full compatibility, GUI, multimedia, or Qt | When you need minimal footprint, simple CLI or control apps |

---

### ✅ Why GLIBC is used in Qt Play Media
- Provides **full API coverage** required by `QtMultimedia`, `GStreamer`, and other multimedia frameworks.  
- Supports **threads, dynamic linking, and plugins** (needed by `QMediaPlayer`, `libgstmediaplayer`, etc.).  
- Ensures **stable symbol versions** (`__ctype_b_loc`, `dlopen`, `pthread_*`) that many codec libraries depend on.  
- Compatible with modern Linux graphics and audio backends (X11, Wayland, ALSA, PulseAudio).  
- Guarantees that Qt applications can **play video/audio reliably** without missing symbol or runtime crashes.

- In ../luckfox-pico/sysdrv/cfg/cfg.mk

```sh
ifeq ($(SYSDRV_CROSS),arm-none-linux-gnueabihf)

ARCH := arm
SYSDRV_CROSS_CFLAGS += -march=armv7-a -mfpu=neon -mfloat-abi=hard
endif
```

- In ..//luckfox-pico/sysdrv/tools/board/toolkits/Makefile

Add sysroot of gblic toolchain
```sh
# ✅ sysroot của toolchain glibc
SYSDRV_SYSROOT ?= /home/bo/linux/Luckfox/luckfox-pico/tools/linux/toolchain/arm-none-linux-gnueabihf/arm-none-linux-gnueabihf/libc

all:
	rm -rf $(PKG_BIN)/io;
	mkdir -p $(CURRENT_DIR)/$(PKG_BIN)/usr/bin;
	$(SYSDRV_CROSS)-gcc --sysroot=$(SYSDRV_SYSROOT) $(PKG_CONF_OPTS) io.c -o $(PKG_BIN)/usr/bin/io
	$(SYSDRV_CROSS)-strip -s $(PKG_BIN)/usr/bin/io
	$(call MAROC_COPY_PKG_TO_SYSDRV_OUTPUT, $(SYSDRV_DIR_OUT_ROOTFS), $(CURRENT_DIR)/$(PKG_BIN))
```

### 3.2 Create QT project media

1. Create QT quick application Empty

- In main.cpp, Add enviroment variable 

```cpp
#include <QGuiApplication>
#include <QQmlApplicationEngine>

int main(int argc, char *argv[])
{
    // --- Thiết lập biến môi trường ---
    qputenv("QT_QPA_FONTDIR", QByteArray("/usr/share/fonts/dejavu/"));
    qputenv("QT_QPA_PLATFORM", QByteArray("linuxfb"));             // hoặc "eglfs" nếu có GPU
    qputenv("QT_QPA_EGLFS_FB", QByteArray("/dev/fb0"));
    qputenv("QT_QPA_EGLFS_WIDTH", QByteArray("800"));
    qputenv("QT_QPA_EGLFS_HEIGHT", QByteArray("480"));
    qputenv("QT_QPA_EGLFS_PHYSICAL_WIDTH", QByteArray("154"));
    qputenv("QT_QPA_EGLFS_PHYSICAL_HEIGHT", QByteArray("86"));
    qputenv("QT_SCALE_FACTOR", QByteArray("1"));
    qputenv("QT_FONT_DPI", QByteArray("96"));
    qputenv("QT_MULTIMEDIA_PREFERRED_PLUGINS", "alsa");
    QCoreApplication::setAttribute(Qt::AA_EnableHighDpiScaling);

    QGuiApplication app(argc, argv);

    QQmlApplicationEngine engine;
    const QUrl url(QStringLiteral("qrc:/main.qml"));
    QObject::connect(&engine, &QQmlApplicationEngine::objectCreated,
                     &app, [url](QObject *obj, const QUrl &objUrl) {
        if (!obj && url == objUrl)
            QCoreApplication::exit(-1);
    }, Qt::QueuedConnection);
    engine.load(url);

    return app.exec();
}
```

- In main.qml, create button play and pause mp3 

```qml
import QtQuick 2.12
import QtQuick.Controls 2.5
import QtMultimedia 5.15

ApplicationWindow {
    visible: true
    width: 800
    height: 480
    title: qsTr("Hello World")
    MediaPlayer {
        id: player
        source: "file:///root/Phepmau.mp3"
    }
    Button {
        x: 262; y: 294; width: 87; height: 70

            text: player.playbackState === MediaPlayer.PlayingState ? "⏸️" : "▶️"
            background: Rectangle {
                color: "transparent"
                border.color: "white"
                radius: 10
            }
            onClicked: {
                if (player.playbackState === MediaPlayer.PlayingState)
                    player.pause()
                else
                    player.play()
            }
            // ✅ Cập nhật icon khi trạng thái thay đổi
            Connections {
                target: player
                onPlaybackStateChanged: parent.text = player.playbackState === MediaPlayer.PlayingState ? "⏸️" : "▶️"
            }
        }
}
```
