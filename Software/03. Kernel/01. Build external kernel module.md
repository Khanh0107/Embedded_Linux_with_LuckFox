# Build external kernel module

## 1. Prerequisites

- Install `sshpass`

```
sudo apt install sshpass
```

* **Cross toolchain** (comes with Luckfox BSP):

  ```
  .../luckfox-pico/tools/linux/toolchain/arm-rockchip830-linux-uclibcgnueabihf/bin/arm-rockchip830-linux-uclibcgnueabihf-
  ```

  > ⚠️ The `CROSS_COMPILE` variable **must end with a dash `-`**.
  > Check version:

  ```bash
  /path/to/toolchain/bin/arm-rockchip830-linux-uclibcgnueabihf-gcc --version
  ```

* **Kernel source**:

  ```
  .../luckfox-pico/sysdrv/source/kernel
  ```

* **Out-of-tree kernel build output**:

  ```
  .../luckfox-pico/sysdrv/source/objs_kernel
  ```

---

## 2. Prepare the kernel (one-time setup)

Edit the board configuration file:
`.../luckfox-pico/project/cfg/BoardConfig_IPC/BoardConfig-EMMC-Buildroot-RV1106_Luckfox_Pico_Ultra_W-IPC.mk`

Update:

```makefile
# Kernel defconfig
# export RK_KERNEL_DEFCONFIG=luckfox_rv1106_linux_defconfig
export RK_KERNEL_DEFCONFIG=luckfox_rv1106_linux_kernel_defconfig
```

> I copied `luckfox_rv1106_linux_defconfig` and renamed it to `luckfox_rv1106_linux_kernel_defconfig`.

Then run:

```bash
cd ../luckfox-pico/sysdrv/source/kernel

# Create .config in O=../objs_kernel
make ARCH=arm CROSS_COMPILE=../../../tools/linux/toolchain/arm-rockchip830-linux-uclibcgnueabihf/bin/arm-rockchip830-linux-uclibcgnueabihf- \
    O=../objs_kernel luckfox_rv1106_linux_kernel_defconfig

# Prepare headers and symbol files for external modules
make ARCH=arm CROSS_COMPILE=../../../tools/linux/toolchain/arm-rockchip830-linux-uclibcgnueabihf/bin/arm-rockchip830-linux-uclibcgnueabihf- \
    O=../objs_kernel modules_prepare
```

> This step creates `.config`, `Module.symvers`, and prepares headers in `objs_kernel` for external module builds.

---

## 3. Write a simple kernel module

Create project structure:

```
hello_demo/
├── hello.c
└── Makefile
```

**hello.c**

```c
#include <linux/module.h>
#include <linux/kernel.h>
#include <linux/init.h>

static int __init hello_init(void)
{
    pr_info("Hello_world: Module loaded!\n");
    return 0;
}

static void __exit hello_exit(void)
{
    pr_info("Hello_world: Module unloaded!\n");
}

module_init(hello_init);
module_exit(hello_exit);

MODULE_LICENSE("GPL");
MODULE_AUTHOR("Tdieney");
MODULE_DESCRIPTION("Simple Hello Module");
MODULE_VERSION("1.0");
```

**Makefile**

```
obj-m += hello.o

KERNEL_SRC := /home/tdieney/workspace/luckfox_workspace/luckfox-pico/sysdrv/source/kernel
PWD ?= $(shell pwd)
MAKE := make
ARCH := arm
CROSS_COMPILE := /home/tdieney/workspace/luckfox_workspace/luckfox-pico/tools/linux/toolchain/arm-rockchip830-linux-uclibcgnueabihf/bin/arm-rockchip830-linux-uclibcgnueabihf-
KBUILD_OUTPUT := $(abspath $(dir $(lastword $(KERNEL_SRC))))/objs_kernel

all:
	make O=$(KBUILD_OUTPUT) -C $(KERNEL_SRC) M=$(PWD) modules \
ARCH=$(ARCH) CROSS_COMPILE=$(CROSS_COMPILE)
	sshpass -p 'luckfox' scp hello.ko root@192.168.1.13:/
	
clean:
	rm -f *.ko *.o *.mod *.mod.o *.mod.c *.symvers *.order *.cmd
```

- **Note:**
  - Update `KERNEL_SRC` and `CROSS_COMPILE` paths in the Makefile to match your environment.
  - The `sshpass` command in the `all` target copies the built module to the board. Adjust the IP address and password as needed.

---

## 4. Build the module

```bash
cd hello_demo
make
```

Expected output:

```
hello_demo/
├── hello.c
├── hello.ko
├── hello.mod.c
├── hello.mod
├── hello.o
├── Makefile
├── modules.order
└── Module.symvers
```

---

## 5. Run on the board

Install module:

```bash
insmod hello.ko
```
