# ADC - Read Potentiometer

This guide demonstrates how to read and convert the value of a potentiometer using the SAR ADC on the RV1106 platform, including device tree configuration and C code integration in U-Boot.

## Step 1: Device Tree Configuration

Add a new node for the potentiometer in your `../luckfox-pico/sysdrv/source/uboot/u-boot/arch/arm/dts/rv1106-luckfox.dts` device tree file:

```dts
potentiometer {
    compatible = "simple-bus";
    io-channels = <&saradc 1>;         // Use SAR ADC channel 1
    io-channel-names = "pot-channel";  // Name for the channel
    status = "okay";                   // Enable the node
    u-boot,dm-spl;                     // U-Boot SPL support
};
```

## Step 2: Implement ADC Reading Function in U-Boot

Add the following function to your `../luckfox-pico/sysdrv/source/uboot/u-boot/board/rockchip/evb_rv1106/evb_rv1106.c` file to read and convert the potentiometer value:

```c
/**
 * Read potentiometer value from ADC1, convert to voltage and percentage.
 */
static int read_potentiometer_adc1(void)
{
    unsigned int adc_raw_value;
    unsigned int voltage_mv;
    int ret;

    printf("***** Reading Potentiometer on ADC1 *****\n");

    // Read ADC channel 1 (SAR ADC)
    ret = adc_channel_single_shot("saradc", 1, &adc_raw_value);
    if (ret) {
        printf("ADC channel 1 read failed (%d)\n", ret);
        return ret;
    }

    printf("ADC read successful\n");

    /*
     * Convert raw ADC value to voltage (mV)
     * - RV1106 SAR ADC: 10-bit resolution (0-1023)
     * - Reference voltage: 1.8V (1800 mV)
     */
    voltage_mv = (adc_raw_value * 1800) / 1023;

    printf("Potentiometer ADC1: Raw=%u, Voltage=%u.%03u V\n", 
           adc_raw_value, voltage_mv/1000, voltage_mv%1000);

    // Calculate potentiometer position as percentage (0-100%)
    int percentage = (adc_raw_value * 100) / 1023;
    printf("Potentiometer position: %d%%\n", percentage);

    return 0;
}
```

### Explanation

- The function reads the raw ADC value from channel 1.
- Converts the raw value to millivolts using the reference voltage (1.8V).
- Prints both the raw value and the calculated voltage.
- Calculates and prints the potentiometer position as a percentage.

## Step 3: Call the Function in Board Initialization

Invoke the function in your board initialization routine, typically in `rk_board_late_init()`:

```c
int rk_board_late_init(void)
{
    // ...existing code...

    // Read potentiometer value during board late init
    read_potentiometer_adc1();

    // ...existing code...
}
```

## Step 4: Configure ADC Driver in U-Boot (menuconfig)

To enable ADC support in U-Boot, follow these steps:
At `../luckfox-pico/sysdrv/source/uboot/u-boot/` path,
1. Run the following command to open the configuration menu:

    ```sh
    make ARCH=arm menuconfig
    ```

2. In the menu, navigate to **Device Drivers**.
    Tick (*) the following options:
    - **Enable ADC drivers using Driver Model**
    - **Enable Rockchip SARADC driver**<br>
    
    Reference image for enabling ADC in menuconfig:
        <p align="center">
        <img src="../99. Images/adc_enable.png"/>
        </p>
3. Save and Exit (Ctrl +C)

4. Generate defconfig file (**savedefconfig**).

    ```sh 
    make ARCH=arm savedefconfig
    ```
5. Move new created file back to /configs folder
    ```sh
    mv defconfig configs/luckfox_rv1106_uboot_custom_defconfig
    ```
6. Back to `../luckfox-pico/`  and run command

    ```sh
        ./build.sh clean uboot
        ./build.sh uboot
    ```
